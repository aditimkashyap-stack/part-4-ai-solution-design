# Part 4: AI Solution Design for a Business Problem
Dataset used:
https://drive.google.com/drive/folders/1akV6po4Nrgkc3yQrJkzA6cJlV-wBvUYs?usp=sharing

## Domain Selected
Healthcare

## Business Problem
Hospitals receive thousands of patient appointment requests, symptom descriptions, and support messages daily. Manually prioritizing urgent cases is slow and error-prone.

## Proposed AI Solution
An NLP-based patient message classification system that automatically categorizes patient messages into:
- Emergency
- High Priority
- Medium Priority
- Low Priority

## AI Task Type
Text Classification

## Recommended Model
Transformer-based NLP model (BERT)

## Expected Benefits
- Faster patient triage
- Reduced response time
- Improved patient satisfaction
- Better healthcare resource allocation
